Gogs中自定义首页

2024/02/23 git docker 共 534 字，约 2 分钟

如果Gogs服务不是在docker中进行部署，可以通过官网文档法进行修改。

当使用docker部署服务时，官网中没有详细说明怎样处理，主要是要找到映射相关文件夹。

比如，在创建docker容器时，有如下映射：

/data:/data

#结构如下
/data
|-- git
|   |-- gogs-repositories
|-- ssh
|   |-- # ssh public/private keys for Gogs
|-- gogs
    |-- conf
    |-- data
    |-- log

其中，/data/gogs就对应自定义目录：custom 文件夹，相对于官网，自定义文件夹结构如下：

自定义文件夹docker映射文件夹备注
custom文件夹/data/gogs 
custom/templates/data/gogs/templates 
custom/public/data/gogs/public 
custom/public/img/favicon.png/data/gogs/public/img/favicon.png自定义站点图标
custom/templates/home.tmpl/data/gogs/templates/home.tmpl自定义首页

模板文件的变更，需要重启docker服务，静态文件的变更不需要。
